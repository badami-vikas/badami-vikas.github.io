# zazoo.me

Marketing site for Zazoo: AI companions for real work. Served by GitHub Pages at https://zazoo.me (CNAME) / https://badami-vikas.github.io.

- `index.html` – Platform landing (interactive companion family)
- `training.html` – Training programs: cohort journey wheel, current + upcoming programs
- `consulting.html` – AI readiness, systems building (agents/workflows/skills/hooks), vision
- `assets/zazoo.css` – design tokens + components (mirrors the Zazoo design-system project on claude.ai/design)
- `assets/zazoo.js` – felt companion SVGs + shared interactions (vanilla, no build step)
- `docs/experience-guide.md` – how to keep the empathy/emotional feel consistent on any surface
- `docs/website-interactions-task.md` – planned follow-up: full interactive redesign + 2.5D animal Zazoos

No build step: edit, commit, push to `main`, Pages deploys.
